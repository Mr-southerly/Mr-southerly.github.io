Cadence之Layout常用快捷键

发表于 2021-10-18 分类于 工具篇
功能键 作用 备注
左键 选中一个图形，如果两个图形交叠的话，单击选中其中一个图形，再单击选中另一个图形，用左键框选，选中一片图形，某个图形要被完全包围才会被选中
中键 放大（相当于Z+左键）
右键 调出常用菜单命令
滚轮 放大缩小版图视图
方向键 放大缩小版图视图
Shift+滚轮 左右移动版图视图
Shift+方向键 鼠标每次移动半个格点距离
Ctrl+方向键 移动cell使版图边界与视图边界对齐
Shift+左键 连续选择多个图像，如果不小心选多了，可以用Ctrl+左键减选图形 Cadence菜单中大写表示+按Shift，Ctrl写成^
F1 显示帮助窗口
F2 保存
F3 Save a Copy 在选取相应工具显示相应属性对话框，可以设置画线的走向，斜角等，比如在选取Path工具后，想要控制Path的走向，可以按F3调出对话框进行设置；还有如果想实现图形的镜像对称，也可以用F3键，选择对话框中的sideway（左右镜像）和updown（上下镜像）
F4 Toggle Partial Select，就是用来控制是否可以部分选择一个图形
F5 Open,打开
F8 Guided Path Create，切换至L90XYFirst
A Quick Align，快速对准
Ctrl+A 全选
B 去某一级（Go to Level）
Shift+B 从子模块版图中返回上层版图
C copy复制
Shift+C chop裁切。首先调用命令，然后选中图形，然后随意裁切画好的图形，按F3有很多设置
D 取消选择。这个也可以用鼠标点击空白区域实现，Shift+D和Ctrl+D也是取消选择
F 满工作区显示
Shift+F 显示所有等级图层。在调用多层Instance时用起来方便
Ctrl+F 显示上层等级图层
G gravity吸i附，开关引力。如果格点设置的不合适或遇到不规则的斜边时使用较方便，将鼠标放到图形的断点或边上时，可使用G。
I 添加元器件，左键放置、右键旋转、Esc推出载入
K ruler，标尺工具，画标尺
Shift+K label，标签工具，标签要加在特定的text层上
M move，移动工具，点选move工具后，选中要移动的图形，然后在屏幕上任意一处单击一下，这个就是确定移动的参考点，然后可以自由移动了。这个也可以通过鼠标先选中一个图像，移动鼠标箭头办成十字方向的时候就可以通过拖动来实现
Shift+M merge，合并工具，合并在一起的图形会变成一个平滑的整体；移动PIN：选中后按键盘的Shift+M，单击右键是旋转
N diagonal，斜45°对角+正交，多次按可以变换走线方向
Shift+N orthogonal，直角正交 Shift+N和Ctrl+N是控制走向的
Ctrl+N 先横后竖，L90XFirst
O 插入接触孔，create contact
Shift+O rotate旋转
P 画path，摁P后点击连接起始端，选择要连接的层，再单击结束端，再摁回车结束
Shift+P polygon，多边形工具
Ctrl+P pin，插入引脚
Q 查看器件属性，左键选中后按Q
Shift+Q 打开设计属性对话框，左键选中后按Shift+Q
R rectangle，矩形工具，画矩形
Shift+R reshape，重定形，就是在原来的图形上再补上一块图形
Ctrl+R redraw，重画
S stretch，框选要拉伸图形，再拉伸
Shift+S search，查找
Ctrl+S split，添加拐点，就是配合stretch命令可以是原来值的path打弯
T tap，查看该点各层信息
Shift+T tree
Ctrl+T Zoom to Set
U undo，撤销
Shift+U 取消撤销
V attatch，关联，将一个子图形（child）关联到一个父图形（parent）后，若移动parent，则child也将跟着移动，若移动child，则parent不会移动。可以将label关联到pad上
Ctrl+V Type in CIW
W 连线
X 保存电路原理图
Shift+X 进入版图的子模块
Ctrl+X fit edit，适合编辑
Y yank，区域复制，如果你想复制某个区域的全部图形而且又不是完整的图形，那么copy就不行的，先按Y，鼠标框选中要复制的区域，然后Shift+Y粘贴即可；copy只能复制完整图形对象
Z 视图放大
Shift+Z 缩小版图视图两倍
Ctrl+Z 放大版图视图两倍
Esc cancel，取消
Tab 平移视图Pan，按Tab，用鼠标点击视图区中某一点，视图就会移至以该点为中心
Delete 删除
Backspace 撤销上一点
Enter 确定图形的最后一点，也可以双击鼠标键结束
